Desafio 9 - DOM - EVENTOS

Document Object Model es una estructura de objetos generada por el navegador que representa la pagina HTML actual. Con JavaScript ocupamos el DOM para acceder a sus elementos y modificarlos de forma dinamica, tanto en su estructura, estilo y contenido. En este modelo cada etiqueta del HTML es un objeto que llamamos nodo -tenemos nodos padre y nodos hijos, segun la anidacion-. Todos esos objetos son accesibles mediante el objeto global document, por ej document.body es nodo que representa a la etiqueta body. Eventos

Es la manera de controlar las acciones de los asuarios -por ej. un click, escribir en un campo, doble click, enter, etc. - y definir el comportamiento de la pagina o aplicacion cuando se produzcan. JavaScript ejecuta una funcion asociada, que esta relacionada a escuchar ese evento: El metodo addEventListener permite definir que evento escuchar sobre cada elemento del HTML.
