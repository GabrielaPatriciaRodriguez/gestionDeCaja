Desafio 6 - Incorporando Array de Objetos

Un array es un objeto primitivo. Es una lista de elementos -puede agrupar string, booleanos, numbers, objetos, y hasta puede agrupar otros arrays. Estos elementos pueden variar en cualquier momentos, agregar, quitar, copiar etc. Los array tienen posiciones, comienzan con la posicion 0, es decir el primer elemento tiene la posicion 0, luego viene la posicion 1, y asi sucesivamente, todos sus elementos tienen un orden. Se declara un array con corchetes.
